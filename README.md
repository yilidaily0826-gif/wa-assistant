# temu poland local to local seller center

这是一个给你做的网页样品（静态原型），参考了你提供的截图风格。

## 功能说明（当前版本）
- 左侧工作台导航
- 顶部项目标题和操作按钮
- 项目卡片展示
- `Active / Negotiating / Archived` 标签切换
- 页面自适应（手机/平板/桌面）

## 如何打开

### 方式 1：直接打开
直接双击 `index.html` 即可在浏览器查看。

### 方式 2：本地启动（推荐）
先确保你安装了 Node.js，然后在项目目录执行：

```bash
npm run start
```

浏览器访问：

```text
http://localhost:5173
```
